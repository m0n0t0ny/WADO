Data Types Primitivi

  • string - insieme di caratteri racchiusi da apici
  • number - numeri senza apici
  • boolean - true / false
  • undefined - assenza di valore
  • null - assenza di valore intenzionale

    • ----------

  • symbol - carattere univoco autogenerato
  • bigInt - per lavorare con numeri molto grandi

Data Types Strutturali

  • object - raccoglitore di chiavi sotto forma di chiave-valore
  • array - una variabile che può contenere molteplici valori ai quali ci si può riferire attraverso un indice
  • function - un blocco di codice che potremo eseguire più volte

Variabili

le variabili possono avere uno scope (raggio di azione) di livello:

  • globale - essere sempre accessibili
  • blocco - essere accessibili solo all'interno del blocco
